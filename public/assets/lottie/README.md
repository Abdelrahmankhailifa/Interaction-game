# Lottie Animations

Place your Lottie animation JSON files here.

## Required Animations (referenced in story.json):

- `tweety-welcome.json` - Welcome animation
- `tweety-flying.json` - Flying animation
- `tweety-curious.json` - Curious animation
- `tweety-excited.json` - Excited animation
- `tweety-happy.json` - Happy animation
- `tweety-celebrating.json` - Celebrating animation
- `tweety-sad.json` - Sad animation
- `tweety-encouraging.json` - Encouraging animation
- `tweety-playing.json` - Playing animation
- `tweety-thinking.json` - Thinking animation
- `tweety-concentrating.json` - Concentrating animation

You can download free Lottie animations from:
- https://lottiefiles.com/
- https://lordicon.com/

Or create your own using After Effects with the Bodymovin plugin.

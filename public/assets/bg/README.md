# Background Images

Place your scene background images here.

## Recommended Specifications

- **Format**: JPG or PNG
- **Size**: 1920x1080 (Full HD) or higher
- **Aspect Ratio**: 16:9 recommended
- **File Size**: Optimize for web (< 500KB recommended)

## Usage

Reference in scene JSON:

```json
{
  "backgroundImage": "/assets/bg/scene1-bg.jpg"
}
```

## Examples

- `garden-sky.jpg` - Garden scene with sky
- `forest-path.jpg` - Forest scene
- `magical-castle.jpg` - Castle scene
